# Liquid Snake

一个带液态玻璃风格 UI 的贪吃蛇小游戏，使用原生 `HTML / CSS / JavaScript` 实现，无需构建工具。

## Features

- 液态玻璃风格界面，包含磨砂卡片、环境光斑和发光棋盘
- 开场欢迎页，支持点击按钮或直接按键进入游戏
- 四档难度：`新手`、`简单`、`普通`、`困难`
- 穿墙模式可切换：新手默认开启，其他难度默认关闭
- 随机限时 Bonus 球，出现后若不吃会自动消失
- 最高分保存在浏览器 `localStorage`
- 支持键盘操作，也支持移动端触控按钮和棋盘滑动

## Demo Gameplay

- 普通食物：提升分数并让蛇变长
- 难度越高：蛇移动越快，普通食物和 Bonus 分值越高
- Bonus 球：随机出现，限时存在，吃到可获得复数分
- 穿墙模式开启时：蛇会从棋盘另一侧穿出
- 穿墙模式关闭时：撞墙直接结束

## Getting Started

这个项目是纯静态页面，直接打开即可运行。

### Option 1: 直接打开

直接双击 `index.html`，或在浏览器中打开它。

### Option 2: 本地静态服务

如果你希望用本地服务访问，可以在项目目录运行任意静态服务器，例如：

```powershell
python -m http.server 8080
```

然后访问 [http://localhost:8080](http://localhost:8080)。

## Controls

### 键盘

- `方向键` / `WASD`：控制移动
- `Space`：暂停 / 继续
- 欢迎页状态下：`Enter` / `Space` 可进入游戏，方向键可直接开局

### 触控

- 点击方向按钮控制移动
- 在棋盘区域滑动控制方向

## Difficulty Rules

| 难度 | 特点 | 穿墙默认状态 |
| --- | --- | --- |
| 新手 | 速度最慢，容错最高，普通食物分值最低 | 开启 |
| 简单 | 比新手更快，得分更高 | 关闭 |
| 普通 | 平衡模式，节奏更紧 | 关闭 |
| 困难 | 速度最快，普通食物和 Bonus 分值最高 | 关闭 |
